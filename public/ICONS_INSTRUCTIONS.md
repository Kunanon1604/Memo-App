# PWA Icons Instructions

To make **PureNote App** installable as a PWA, you need to convert `icon.svg` into two PNG files and place them in this folder.

### 1. Convert SVG to PNG
Use an online tool like [CloudConvert](https://cloudconvert.com/svg-to-png) or [SVGtoPNG.com](https://svgtopng.com/).

### 2. Required Files
Save the converted images with these exact names and sizes:
- **pwa-192x192.png** (Resize to 192x192 px)
- **pwa-512x512.png** (Resize to 512x512 px)

### 3. Verification
Once these two files are in this `public/` folder, the "Install" button will appear in your browser when you run the app.
